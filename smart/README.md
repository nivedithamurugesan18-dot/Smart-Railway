SMART RAILWAY CROWD MANAGEMENT SYSTEM

--------------------------------------------

PROJECT DESCRIPTION

The Smart Railway Crowd Management System is developed using Python OOP concepts and Data Structures. 

The main aim of this project is to monitor and control crowd on railway platforms.

The system helps railway authorities:
- Store passenger details
- Verify tickets
- Manage passenger queue
- Predict waiting time
- Monitor platform crowd
- Generate crowd reports

--------------------------------------------

MODULES USED

MODULE 1 – Passenger Entry Module
- Enter passenger data
- Display passengers
- Search passengers
- Count passengers
- Generate reports

MODULE 2 – Ticket Verification Module
- Verify ticket
- Store ticket data
- Check valid tickets

MODULE 3 – Crowd Analysis Module
- Queue management
- Waiting time prediction
- Crowd monitoring
- Platform crowd analysis
