Smart Railway Crowd Management System

Project Overview

The Smart Railway Crowd Management System is a Python-based project developed to monitor and manage crowd levels on railway platforms. The system helps railway staff store passenger details, verify tickets, count platform crowd, and identify overcrowded areas. Based on the number of people on the platform, the system classifies the crowd as Safe Zone, Moderate Crowd, or High Risk Zone.

This project is developed using Python with Object-Oriented Programming (OOP) concepts, data structures, and collections.


---

Problem Statement

Railway platforms become overcrowded during peak hours and festival seasons. Large crowds create safety risks, long queues, confusion, and difficulties in crowd control. Manual monitoring of passengers is time-consuming and less effective.

To solve this problem, this project helps:

Monitor passenger crowd

Verify tickets

Analyze crowd level

Generate crowd reports

Improve passenger safety

Crowd Analysis

People Count	Status

Less than 500	Safe Zone
500 - 699	Moderate Crowd
700 and above	High Risk


If the crowd becomes too high, the system warns railway staff not to issue more platform tickets.
